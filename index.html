Sincronización & Configuración

🔄 Sincronización con Google Sheets No configurado

Google Sheets actúa como base de datos compartida. Todos los docentes ven los mismos datos cuando hay internet. Sin internet todo funciona local y sincroniza al reconectar.

Está en la URL de tu Sheet: docs.google.com/spreadsheets/d/[ESTE ID]/edit

En Google Cloud Console → Habilitar Sheets API → Credenciales → Crear API Key

📋 Guía para configurar Google Sheets
1
Crea una hoja en Google Drive: "Sistema I.E. Las Cruces"
2
Copia el ID de la URL (la parte larga entre /d/ y /edit)
3
Haz la hoja pública: Compartir → Cualquier persona con el enlace → Ver
4
En console.cloud.google.com → Activa Google Sheets API → Crea un API Key
5
Pega los datos arriba, guarda y haz clic en "Sincronizar ahora"
